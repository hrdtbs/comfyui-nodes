# StringTools

文字列操作を行うためのユーティリティノード群です。
プロンプトの結合や置換、切り出しなどに使用できます。

## 含まれるノード

### String Concatenate
2つの文字列を指定した区切り文字で結合します。

- **入力**:
    - `text_a`: 最初の文字列
    - `text_b`: 2番目の文字列
    - `separator`: 区切り文字（デフォルトは空文字）
- **出力**:
    - `result`: 結合された文字列

### String Replace
文字列内の特定の文字列を別の文字列に置換します。

- **入力**:
    - `text`: 対象の文字列
    - `old`: 置換前の文字列
    - `new`: 置換後の文字列
- **出力**:
    - `result`: 置換後の文字列

### String Slice
文字列を指定した開始位置と終了位置で切り出します（スライス）。
Pythonのスライス操作と同様の動作をします。

- **入力**:
    - `text`: 対象の文字列
    - `start`: 開始インデックス（デフォルト: 0）
    - `end`: 終了インデックス（デフォルト: 0）。0の場合は文字列の末尾まで切り出します。
- **出力**:
    - `result`: 切り出された文字列

### String Regex Replace
正規表現を使用して文字列を置換します。

- **入力**:
    - `text`: 対象の文字列
    - `pattern`: 正規表現パターン
    - `replacement`: 置換文字列（後方参照 `\1` なども使用可能）
- **出力**:
    - `result`: 置換後の文字列

### String Split
文字列を指定した区切り文字、または改行で分割し、リストとして返します。
次のノードに接続すると、リスト内の各要素に対して処理が実行されます（バッチ処理）。

- **入力**:
    - `text`: 対象の文字列
    - `separator`: 区切り文字（デフォルト: `,`）。`split_mode` が `separator` でかつ空文字の場合は空白で分割します。
    - `split_mode`: 分割モード（デフォルト: `newline`）
        - `newline`: 改行コード (`\n`, `\r`, `\r\n`) で分割します。空行は自動的に除外されます。
        - `separator`: 指定した `separator` で分割します。
- **出力**:
    - `strings`: 分割された文字列のリスト

### String Join
文字列のリストを指定した区切り文字で結合し、1つの文字列にします。
`String Split` の逆の操作や、バッチ処理された文字列をまとめるのに使用します。

- **入力**:
    - `strings`: 対象の文字列リスト（バッチ入力）
    - `separator`: 区切り文字（デフォルト: `\n`）。`\n` と入力すると改行として扱われます。
- **出力**:
    - `string`: 結合された文字列

### String Case
文字列の大文字・小文字を変換します。

- **入力**:
    - `text`: 対象の文字列
    - `case_method`: 変換方法
        - `upper`: すべて大文字にします。
        - `lower`: すべて小文字にします。
        - `title`: 各単語の先頭を大文字にします。
        - `capitalize`: 最初の文字だけ大文字にします。
        - `swapcase`: 大文字と小文字を反転させます。
- **出力**:
    - `result`: 変換後の文字列
