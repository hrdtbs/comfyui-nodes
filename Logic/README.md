# Logic

ComfyUI用の論理演算ノードセットです。
条件分岐やパラメータ制御のために、ブール値の生成、操作、比較を行います。

## ノード一覧

### Logic Boolean
定数のブール値を出力します。

- **入力**:
    - `value`: 真偽値（True/False）
- **出力**:
    - `value`: 設定されたブール値

### Logic Not
ブール値を反転します。

- **入力**:
    - `value`: ブール値
- **出力**:
    - `result`: 反転されたブール値（True -> False, False -> True）

### Logic Operation
2つのブール値に対して論理演算を行います。

- **入力**:
    - `op`: 演算子 (AND, OR, XOR, NAND, NOR, XNOR)
    - `a`: 1つ目のブール値
    - `b`: 2つ目のブール値
- **出力**:
    - `result`: 演算結果

### Logic Compare
2つの入力値を比較します。任意の型（数値、文字列など）を受け付けます。

- **入力**:
    - `op`: 比較演算子 (==, !=, <, >, <=, >=)
    - `a`: 1つ目の値
    - `b`: 2つ目の値
- **出力**:
    - `result`: 比較結果（ブール値）
    - ※ 型が互換性のない場合（例: 文字列と数値の大小比較）は False を返します。
