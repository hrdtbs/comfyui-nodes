# ImageTools

画像処理や解像度計算に関連するユーティリティノード群です。

## 含まれるノード

### Image Properties
画像（Image Tensor）から幅、高さ、アスペクト比、バッチサイズを取得します。

- **入力**:
    - `image`: 画像（IMAGE型）
- **出力**:
    - `width`: 幅（整数）
    - `height`: 高さ（整数）
    - `aspect_ratio`: アスペクト比（浮動小数点数）
    - `batch_size`: バッチサイズ（整数）

### Image Resize Calculator
指定した幅と高さに対してスケールファクターを適用し、新しい寸法を計算します。
アップスケーリング時の解像度計算などに便利です。

- **入力**:
    - `width`: 元の幅（整数）
    - `height`: 元の高さ（整数）
    - `scale_factor`: 倍率（浮動小数点数、デフォルト: 1.0）
- **出力**:
    - `width`: 計算後の幅（整数）
    - `height`: 計算後の高さ（整数）
