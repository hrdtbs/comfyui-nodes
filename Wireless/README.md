# Wireless

ワイヤレス接続機能を提供するノードです。ComfyUIのワークフロー上で、長い接続線を引くことなくデータを転送できます。

## ノード一覧

### Wireless

このノードに入力されたデータは、ワークフロー内の**未接続（Unconnected）**かつ**同じ型（Type）**の入力スロットに自動的に接続されます。
接続処理はJavaScript（フロントエンド）によって行われます。

**機能概要**
1. `Wireless`ノードに入力を接続します（例: `VAE`）。
2. ワークフロー内の他のノード（例: `KSampler`）の `vae` 入力が未接続の場合、自動的にこの`Wireless`ノードからの入力が適用されます。
3. 優先順位:
   - 入力名が一致する場合（例: `vae` -> `vae`）
   - 同じ型の`Wireless`ノードが複数ある場合、最後に処理されたもの（通常はグラフの下/右にあるもの）が優先されます。

**入力 (Inputs)**
* `data` (AnyType): 転送したいデータ。任意の型を受け付けます。

**出力 (Outputs)**
* なし（ダミー出力のみ）

**注意**
* この機能は強力ですが、意図しない接続が発生する可能性があります。使用する際は、意図しない入力が未接続のままになっていないか確認してください。
