# Wireless Nodes

ワークフロー内でワイヤー接続を減らし、スッキリさせるためのノードです。

## 概要
このモジュールは `js/wireless.js` のフロントエンド拡張機能と連携して動作します。
`Wireless` ノードに入力を接続すると、その入力データは同じ型を持つ、まだ接続されていない入力端子へ自動的に供給されるようになります（フロントエンドのスクリプトによって処理されます）。

## ノード一覧

### Wireless
任意の型のデータを受け取り、それをブロードキャストするためのノードです。

#### 入力 (Inputs)
- **data** (Any): 任意のデータ。ここに接続されたデータが、ワークフロー内の他の場所で使用可能になります。

#### 出力 (Outputs)
- なし (内部的に処理されます)
